# /public/models/

Place your 3D model file here:

- **boombox.glb** — The GLTF Binary model that will be rendered when the AR target is detected.

The AR scene in `/src/app/ar/retro-player/ARRetroPlayer.tsx` references this file at `/models/boombox.glb`.

## Getting a test model

You can download a free GLB model from:
- https://market.pmnd.rs/
- https://sketchfab.com/3d-models?type=downloadable&sort_by=-likeCount

Just rename it to `boombox.glb` and place it in this directory.
